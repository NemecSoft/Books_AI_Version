# 《金瓶梅》第73回分析报告

## 回目标题
潘金莲不愤忆吹箫 西门庆新试白绫带

## 主要情节概述
本回主要讲述了以下几个情节：
1. 西门庆在藏春坞看泥水匠打地炕，收到帅府周爷等人送来的分资
2. 潘金莲为西门庆制作白绫带儿，并装入颤声娇药末儿
3. 薛姑子送来安胎气的衣胞符药，建议潘金莲在壬子日空心服用
4. 孟玉楼生日宴上，西门庆因思念李瓶儿而让小优儿唱《忆吹箫》
5. 潘金莲对此不满，与西门庆发生争执
6. 西门庆与应伯爵、温秀才、吴大舅在前边饮酒，展示何太监送的飞鱼蟒衣
7. 潘金莲偷听西门庆与春梅在房中嬉戏
8. 潘金莲回到房中，用白绫带与西门庆云雨之欢

## 人物形象分析

### 潘金莲
- **心机深沉**：为了与西门庆欢好，精心制作白绫带并装入药末
- **嫉妒心强**：对西门庆思念李瓶儿极为不满，言语尖刻
- **善于察言观色**：能敏锐察觉西门庆的情绪变化
- **泼辣直率**：在宴席上直接表达对李瓶儿的不满
- **欲望强烈**：主动与西门庆亲近，毫不掩饰

### 西门庆
- **多情善感**：对已故的李瓶儿仍念念不忘
- **喜好炫耀**：向众人展示何太监送的飞鱼蟒衣
- **善于交际**：与应伯爵等人饮酒谈笑
- **欲望旺盛**：与潘金莲、春梅都有暧昧关系
- **情绪易变**：因潘金莲言语过激而恼怒

### 薛姑子
- **善于察言观色**：了解潘金莲的心思，主动送符药
- **圆滑世故**：懂得迎合施主的心理需求
- **有心计**：通过符药和锦香囊来获取利益

## 社会文化背景
本回反映了明代社会的一些特点：
1. **宗教信仰**：通过薛姑子送符药的情节，展现了当时人们对佛教和符咒的信仰
2. **社会等级**：西门庆与官员交往，体现了商人与官府的关系
3. **女性地位**：潘金莲等女性在家庭中的地位和作用
4. **商业文化**：西门庆的商业活动和人际关系网络

## 文学手法分析
1. **对比手法**：通过潘金莲与李瓶儿的对比，突出人物性格差异
2. **细节描写**：对白绫带制作过程的细致描写，体现了潘金莲的心机
3. **对话描写**：通过人物对话展现性格特点和内心活动
4. **象征手法**：白绫带象征着潘金莲对西门庆的控制欲

## 主题思想
1. **欲望与情感**：探讨人性中的欲望和复杂情感关系
2. **嫉妒与争宠**：表现了女性在男权社会中的生存状态
3. **生死离别**：通过李瓶儿之死，反映人生的无常
4. **权力关系**：揭示了家庭内部的权力结构和人际关系

## 现代意义
1. **人性探讨**：作品对人性的复杂描写具有永恒的价值
2. **社会批判**：对当时社会风气的批判仍有现实意义
3. **女性意识**：潘金莲等女性形象反映了女性的觉醒和抗争
4. **家庭伦理**：对家庭关系和伦理道德的思考

## 语言特色
1. **生动形象**：人物对话生动自然，富有生活气息
2. **讽刺幽默**：通过讽刺手法揭示人物性格和社会现象
3. **细腻描写**：对人物心理和行为的描写细致入微
4. **方言俗语**：运用了大量的方言俗语，增强了作品的地方色彩

## 伏笔设置
1. **潘金莲的白绫带**：为后续情节发展埋下伏笔
2. **薛姑子的符药**：暗示了人物的命运走向
3. **西门庆对李瓶儿的思念**：反映了人物内心的复杂情感

## 重要情节分析
1. **制作白绫带**：体现了潘金莲的心机和对西门庆的控制欲
2. **唱《忆吹箫》**：引发潘金莲的不满，推动情节发展
3. **飞鱼蟒衣展示**：展现了西门庆的炫耀心理和社会地位
4. **云雨之欢**：是本回的高潮，也是人物关系的重要体现

## 总结
第73回通过潘金莲制作白绫带、西门庆思念李瓶儿等情节，深入刻画了人物性格，展现了复杂的人际关系和情感纠葛。作品通过细腻的描写和生动的对话，揭示了人性的复杂和社会的现实，具有很高的文学价值和现实意义。
