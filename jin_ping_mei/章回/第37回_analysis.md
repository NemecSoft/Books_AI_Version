# 《金瓶梅》第37回分析报告

## 回目标题
冯妈妈说嫁韩爱姐 西门庆包占王六儿

## 主要情节概述
本回主要讲述了西门庆通过冯妈妈为蔡太师府中的翟管家寻找妾室，最终选中了韩道国的女儿韩爱姐，并安排将其送往东京。随后，西门庆又通过冯妈妈与韩爱姐的母亲王六儿勾搭成奸的故事。

## 人物形象分析

### 西门庆
- **权势与欲望的结合体**：西门庆利用自己的权势和财富，既为蔡太师府中的翟管家寻找妾室，又趁机满足自己的欲望。
- **狡诈与虚伪**：为了接近王六儿，他谎称是为蔡太师府中的翟管家寻找妾室，实际上是为了自己。
- **贪婪与放纵**：在与王六儿的交往中表现出极度的贪婪和放纵。

### 王六儿
- **风情万种**：作为韩道国的妻子，王六儿被描绘得极具风情，善于打扮，举止妖娆。
- **现实与无奈**：为了家庭利益，她愿意将自己的女儿嫁给别人，同时也为了生活需要而出卖自己。
- **主动与迎合**：在与西门庆的关系中表现出主动迎合的态度。

### 韩爱姐
- **年轻貌美**：作为本回中被选中的女子，韩爱姐被描述为十五岁，生得美貌。
- **命运的牺牲品**：她的婚姻完全由父亲和西门庆安排，体现了当时女性地位的低下。

### 冯妈妈
- **媒婆角色**：作为中间人，冯妈妈在西门庆与王六儿之间起到了牵线搭桥的作用。
- **善于察言观色**：能够准确把握双方的需求和心理。
- **贪图小利**：从西门庆处获得银两作为酬劳。

1. **婚姻制度**：女性的婚姻完全由家长安排，个人意愿不被考虑。
2. **社会阶层**：西门庆作为地方豪富，可以随意安排他人的命运。
3. **商业文化**：通过韩道国开绒线铺的背景，展现了当时的商业活动。
4. **道德沦丧**：通过西门庆与王六儿的不正当关系，揭示了当时社会道德的沦丧。
