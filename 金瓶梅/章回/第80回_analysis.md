# 《金瓶梅》第80回分析报告

## 回目标题
潘金莲售色赴东床 李娇儿盗财归丽院

## 主要情节概述
本回主要讲述了以下几个情节：
1. 西门庆死后首七，应伯爵等七人凑钱祭奠
2. 李娇儿和王六儿前来吊唁，月娘对王六儿态度恶劣
3. 潘金莲与陈敬济在西门庆灵前调情
4. 西门庆出殡，李娇儿与吴二舅有私情被发现
5. 月娘与李娇儿发生冲突，最终将其打发回院
6. 蔡御史前来吊唁，送上赙仪
7. 李娇儿被张二官娶为二房
8. 应伯爵向张二官推荐潘金莲

## 人物形象分析

### 潘金莲
- **放荡不羁**：西门庆死后不久就与陈敬济调情
- **善于诱惑**：主动勾引陈敬济，表现出强烈的欲望
- **心机深沉**：在西门庆灵前仍不忘调情
- **争强好胜**：在家庭中试图占据主导地位

### 李娇儿
- **势利现实**：西门庆一死就急于离开，回归本行
- **贪图钱财**：与吴二舅有私情，偷盗财物
- **善于钻营**：通过应伯爵的关系嫁入张二官家
- **虚荣心强**：追求更好的生活条件

### 吴月娘
- **端庄稳重**：作为正妻，在家庭中保持威严
- **有原则**：对李娇儿的行为表示不满
- **坚强勇敢**：在西门庆死后承担起家庭重任
- **传统守旧**：对潘金莲的行为表示不满

### 应伯爵
- **势利小人**：西门庆一死就转投张二官门下
- **善于钻营**：为张二官出谋划策，推荐潘金莲
- **圆滑世故**：在复杂的人际关系中游刃有余
- **忘恩负义**：背叛了与西门庆的友情

### 陈敬济
- **轻浮放荡**：西门庆死后不久就与潘金莲调情
- **缺乏道德约束**：在长辈灵前仍不忘私情
- **年轻冲动**：容易被美色诱惑

## 社会文化背景
本回反映了明代社会的一些特点：
1. **社会风气**：通过人物的行为反映了当时社会的道德沦丧
2. **女性地位**：女性在家庭和社会中的地位和作用
3. **商业文化**：通过应伯爵等人的行为，展现了当时的商业活动
4. **官场关系**：通过蔡御史吊唁的情节，体现了官场的人情世故

## 文学手法分析
1. **对比手法**：通过西门庆生前的繁华与死后的凄凉形成对比
2. **细节描写**：对人物行为和心理的细致描写，增强了真实感
3. **对话描写**：通过人物对话展现性格特点和内心活动
4. **讽刺手法**：通过讽刺揭示人物性格和社会现象

## 主题思想
1. **道德沦丧**：通过人物的行为揭示道德沦丧的社会现象
2. **人情冷暖**：通过应伯爵等人的行为，揭示了人情的冷漠
3. **权力关系**：揭示了社会中的权力结构和人际关系
4. **女性命运**：通过李娇儿和潘金莲的命运，反映了女性在封建社会中的处境

1. **人性探讨**：作品对人性的复杂描写具有永恒的价值

4. **道德伦理**：对家庭关系和伦理道德的思考

2. **讽刺幽默**：通过讽刺手法揭示人物性格和社会现象

1. **潘金莲与陈敬济的调情**：展现了人物的放荡和道德沦丧
2. **李娇儿被打出家门**：反映了家庭内部的矛盾和冲突
3. **应伯爵的背叛**：揭示了人情的冷漠和势利
4. **张二官对潘金莲的兴趣**：为后续情节发展奠定基础

## 总结
第80回通过西门庆死后家庭内部的变化，深刻揭示了人性的复杂和社会的现实。作品通过细腻的描写和生动的对话，展现了人物在权力和金钱面前的真实面目，具有很高的文学价值和现实意义。这一回标志着西门庆时代的结束和新时代的开始，为后续情节的发展奠定了基础。
