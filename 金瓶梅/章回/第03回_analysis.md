# 《金瓶梅》第三回分析报告

## 回目标题
定挨光王婆受贿 设圈套浪子私挑

## 主要情节概述

本回主要讲述了王婆为西门庆设计勾引潘金莲的详细计划，以及西门庆按照计划逐步实施的过程。

### 核心事件线：
1. 西门庆向王婆坦白心事，请求帮助
2. 王婆提出"挨光"五件事的理论
3. 王婆制定详细的勾引计划
4. 西门庆准备绸绢绵子等物品
5. 王婆以做衣服为由邀请潘金莲到家中
6. 西门庆按计划出现在王婆家中与潘金莲见面
7. 三人共饮，西门庆与潘金莲互相倾心

## 人物形象分析

### 王婆
- **社会角色**：媒婆、马百六，善于察言观色
- **性格特征**：狡猾、贪财、善于设计
- **行为表现**：
  - 提出"挨光"五件事的理论
  - 制定详细的勾引计划
  - 巧妙安排西门庆与潘金莲的见面
- **语言特色**：善于使用比喻和俗语，口才极佳

### 西门庆
- **性格特征**：好色、善于算计、愿意花钱
- **行为表现**：
  - 毫不犹豫地答应王婆的要求
  - 准备绸绢绵子等物品
  - 按照计划精心打扮出现
  - 在酒席上与潘金莲眉来眼去
- **心理活动**：对潘金莲的强烈欲望，急于得手

### 潘金莲
- **性格特征**：风流、善于针线、有心计
- **行为表现**：
  - 主动答应为王婆做衣服
  - 在西门庆面前表现出羞涩
  - 饮酒后春心萌动
- **心理活动**：对西门庆的好感逐渐加深

### 武大郎
- **性格特征**：老实、愚钝、信任他人
- **行为表现**：
  - 对妻子的行为毫无察觉
  - 嘱咐妻子要与王婆搞好关系
- **作用**：作为背景人物，为西门庆和潘金莲的勾当提供便利

## 社会文化背景

### "挨光"文化
1. "挨光"五件事的详细解释
2. 当时社会对偷情行为的认知
3. 媒婆在社会中的特殊地位

### 女性技能
1. 针线活在女性生活中的重要性
2. 女性需要掌握的多种技能（弹唱、棋艺等）
3. 女性在家庭中的地位和作用

### 商业交易
1. 银钱交易在人际关系中的作用
2. 小商贩的生存状态
3. 金钱与人情的关系

## 文学手法分析

### 理论阐述
- 王婆详细阐述"挨光"五件事的理论
- 通过对话形式展现人物性格
- 为后续情节发展做理论铺垫

### 计划描写
- 详细描述王婆的勾引计划
- 分步骤展现计划的实施过程
- 通过具体细节增强真实感

### 对比手法
- 西门庆的急切与潘金莲的羞涩对比
- 王婆的精明与武大郎的愚钝对比
- 西门庆的富有与王婆的贪财对比

### 象征手法
- "绸绢绵子"象征西门庆的诚意和财富
- "酒食"象征情感的催化剂
- "咳嗽为号"象征秘密联络的方式

## 主题思想

1. **欲望与算计**：通过王婆的计划展现人性中的欲望和算计
2. **金钱与人情**：揭示金钱在人际关系中的重要作用
3. **女性的困境**：潘金莲对现状的不满和对改变的渴望
4. **道德沦丧**：通过王婆等人物展现社会道德的沦丧

## 现代意义

本回反映了明代社会的一些重要特征：
- 商业文化对传统道德的冲击
- 女性在封建社会中的困境和反抗
- 人际关系中的功利主义倾向
- 社会对不道德行为的默许

## 语言特色

1. 大量使用俗语和比喻，增强表达效果
2. 对人物对话的细致描写
3. 使用诗词增强文学性
4. 口语化表达增强了作品的生活气息

## 伏笔设置

1. 王婆的详细计划为后续情节发展埋下伏笔
2. 西门庆与潘金莲的相互倾心为私情发展做铺垫
3. 武大郎的愚钝为将来悲剧埋下隐患

## 重要概念解析

### "挨光"五件事
1. **潘**：潘安的貌（外貌）
2. **驴**：驴子的精力（性能力）
3. **邓**：邓通的钱财（财富）
4. **小**：青春年少（年龄）
5. **闲**：闲工夫（时间）

这五件事构成了成功"挨光"的必要条件，体现了当时社会对偷情行为的复杂认知。
